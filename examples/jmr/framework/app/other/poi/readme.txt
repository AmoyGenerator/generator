参考网址：http://poi.apache.org/
Apache POI是Apache软件基金会的开放源码函式库，POI提供API给Java程式对Microsoft Office格式档案读和写的功能。
结构： 　　
Excel (SS=HSSF+XSSF) 
Word (HWPF+XWPF) 
PowerPoint (HSLF+XSLF) 
OpenXML4J (OOXML) 
OLE2 Filesystem (POIFS) 
OLE2 Document Props (HPSF) 
Outlook (HSMF) 
Visio (HDGF) 
Publisher (HPBF)
