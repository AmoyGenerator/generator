运行

步骤：

  • 勾选需要运行的任务；
  • 点击"运行"。

运行视图

状态

有三种状态:

  • 完成：任务正常完成。
  • 错误任务：任务配置有错误，请检查任务是否有配置错误。
  • 运行错误：任务运行时错误，在问题中查看和定位错误信息。

问题

运行错误时，会把错误信息显示出来。

例子：在模板中有1/0的运行时错误

运行任务，出现运行时错误，点击查看。

可以看到错误信息，点击链接。

定位到错误的位置，修改后重新运行，直到解决问题为止。

打开

点击打开可以查看生成的文件。

前后比较

当任务模式为合并或覆盖，点击“打开比较”可以比较运行前后文件内容。

撤销

任务模式为新建时，会删除新建的文件。任务模式为合并或者覆盖时，文件内容回滚为运行前。
