xml:for - xml循环迭代

<xml:for>

循环遍历Xml节点

说明

遍历子Xml节点，使用Xpath表达式得到需要遍历的节点。

属性

select（必须）
使用Xpath表达式得到Xml循环的节点

var（必须）
存储集合的遍历对象

status（可选）
设置变量，存储循环信息
符号    描述
count   循环总次数
index   当前循环索引
isFirst 当前循环是否为头索引
isLast  当前循环是否为尾索引

trim（可选）
是否保留输出值的前后空白字符

属性
false:保留输出值的前后空白字符（默认）
true:删除输出值的前后空格（包括换行符）

delimiter（可选）
循环输出之间的分隔符

例子位置：

  • demo/tag/tag.jmr.xml 任务中的 xml/for/for

delimiter使用方法

delimiter循环输出之间的分隔符，如果指定了分隔符，则输出元素间只按分隔符分割，而不在按原来的格式。
