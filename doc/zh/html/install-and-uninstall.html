安装

Eclipse市场

在Eclipse菜单栏中，点击Help -> Eclipse Marketplace，搜索Jmr，点击安装。

Eclipse地址

在Eclipse菜单栏中，点击Help -> Install New Software -> Add

勾选Basic、Jet和Jmr，点击Next直到下载安装完毕并重启Eclipse

如果安装比较慢或者卡住无法安装，请把Contact all update...勾除，并重新安装。

安装过程

通过以上2种方式安装成功后，重启后会提示开始初始化，点击是，开始初始化Jmr组件

安装完成，同时打开本地文档，包含了所有Jmr的教程。

可以看到工具栏中Jmr的图标

注意事项

1.Help里没有Install New Software选项

在菜单栏中选择Window（窗口）-->Preferences（首选项）-->General（常规）-->Capabilities（功能），勾选Classic Update，点击Apply（应用），重启。

2.安装的时候卡住，请勾掉Contact all update...选项。

3.如果出现以下页面，请选择Update my installation the same...

4.安装后，如果出现以下的错误或者其他编译错误。

解决方法：
1）点击以下按钮，重新编译JET2Project。

2）清理或者更换工作空间后重启Eclipse。

卸载

点击工具栏中的卸载

点击是，开始卸载

要彻底删除Jmr插件，请点击是

点击确认

选中Jet和Jmr组件，点击Uninstall完成最后卸载，并重启Eclipse即可
