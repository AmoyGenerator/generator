接下来，通过设置预设方案(plan)来讲解，在设置预设方案之前，我们要先设置“映射和转换”。

映射和转换

首先我们先设置“映射”，这里我们设置了“fullType”和“type”。
fullType：用于设置完成类型，如“java.lang.Integer”。

type：用于设置类型，如“Integer”。

设置“转换”，这里我们设置了“fieldConvertAction”和“propertyConvertAction”。
fieldConvertAction：用于把数据库的字段转换城java程序的格式，如“regist_time”转换为“registTime”。
